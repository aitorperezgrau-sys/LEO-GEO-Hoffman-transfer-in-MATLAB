# LEO-GEO-Hoffman-transfer-in-MATLAB: In-Space Propulsion Feasibility Analysis

# Overview
This repository contains a MATLAB script developed to analyze the feasibility of various propulsion technologies for a bidirectional LEO-to-GEO mission.

## Technologies Evaluated
* Cold Gas CO2
* Hydrazine
* LOX/CH4
* LOX/H2

## key features included
* **Mass vs. Specific Impulse Modeling:** Calculates the required propellant mass across seven orders of magnitude.
* **Data Visualization:** Generates log-scale comparative plots and annotated constraint overlays to easily compare technologies against mission constraints.
* **Business Conclusion:** The model mathematically demonstrates that no current single-stage technology meets the strict mass requirements, validating the commercial need for high-performance green propulsion systems.

## Files Included
* `m_propelente_vs_impulso_especifico.m` - The main MATLAB script containing the mathematical model and plotting functions.
* `propulsion_graphs.pdf` - The exported visualizations generated by the script.

## Skills Demonstrated
MATLAB, Orbital Mechanics, Cost/Mass Modeling, Data Visualization.
